# MSc Szoftech – Card Games

Full stack projekt Flask backenddel és Angular frontenddel.
A rendszer célja kártyajáték módok megjelenítése és valós idejű kommunikáció WebSocketen keresztül.

## Projekt struktúra

- `backend/` – Flask API + WebSocket szerver
- `frontend/` – Angular kliensalkalmazás

## Funkciók

- Felhasználó API (létrehozás, lekérdezés, törlés)
- WebSocket kapcsolat kliens és szerver között
- Játékmód választó kezdőoldal (Spicy, Uno, Blackjack)

## Technológiák

- **Backend**: Flask, websockets, Python 3
- **Frontend**: Angular 21, TypeScript, Vitest
